# Naprendszer
## A Főoldalról
A főoldalon a naprendszer érdekességeiről, alap tudnivalóiról, kutatóiról, űrhajósairól lesz szó. A naprendszer 8 bolgyóinak jellemzése is megtalálható lesz a kezdőlapon.
